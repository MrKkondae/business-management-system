# ADR-001 단일 표준용어 카탈로그

## 상태

승인

## 일자

2026-07-13

## 배경

표준용어를 업무영역별 CSV로 분리하면 다른 영역에서 같은 용어를 사용할 때 행을 복제하기 쉽다. 복제된 용어는 설명, 컬럼명과 도메인이 서로 달라질 수 있으며 전체 표준용어를 조회하려면 별도의 통합 산출물이 필요하다.

표준용어는 시스템 전체에서 동일한 의미와 명칭을 제공하는 전역 데이터 표준이므로 하나의 원본이 필요하다. 업무영역 정보는 파일 분리가 아니라 명시적인 속성으로 관리할 수 있다.

## 결정

- 표준용어의 유일한 원본은 `01.standard/standard-terms.csv`로 한다.
- `owner_area` 열은 용어의 의미와 변경을 책임지는 소유 업무영역을 나타낸다.
- 여러 업무영역에서 사용하는 용어도 한 건만 등록한다.
- 용어 사용 위치는 엔터티 속성과 향후 테이블·컬럼 정의에서 확인하며 표준용어 원본에 사용 영역을 중복 기록하지 않는다.
- 같은 컬럼명을 사용하더라도 한글명과 업무 의미가 다르면 별도 표준용어로 등록할 수 있다.
- 기존 `source_no`는 유지하고 제거한 중복 행의 번호는 재사용하지 않는다.

## 영향

- 업무영역별 `standard-terms-{business-area}.csv`는 제거한다.
- 전체 표준용어를 별도 생성 없이 단일 CSV에서 조회할 수 있다.
- 검증기는 `owner_area` 허용값, 전역 `source_no`, 전역 중복 용어와 도메인 참조를 검사한다.
- 표준용어는 시스템관리부터 업무영역 순서대로 검토하고 보완한다.
