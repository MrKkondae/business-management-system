# ADR-010 1차 데이터베이스 구축 범위

## 상태

승인

## 일자

2026-07-22

## 배경

개발 데이터베이스와 사용자별 접속 계정 구성이 완료되어 애플리케이션 개발에 필요한
초기 스키마를 구축해야 한다. 현재 시스템관리, 공통, 고객관리, 영업관리, 인력관리와
v1.0 최소 프로젝트의 물리 모델이 작성되어 있으며 계약관리, 예산·원가관리와
매출·수금관리의 물리 모델은 아직 작성되지 않았다.

## 결정

1차 데이터베이스 구축에는 물리 모델이 확정된 다음 6개 업무영역의 47개 테이블을 포함한다.

- 시스템관리: 14개
- 공통: 4개
- 고객관리: 3개
- 영업관리: 12개
- 인력관리: 13개
- 프로젝트관리: 1개(`TB_PRJ_MST`)

테이블, 컬럼, 제약조건과 인덱스는 각 업무영역의 물리 모델 CSV를 기준으로 Flyway
버전 마이그레이션을 작성해 생성한다.

## 제외 범위

- 계약관리
- 예산·원가관리
- 매출·수금관리
- 프로젝트 일정, 이슈, 위험, 산출물, 진척률, 원가와 손익

## 영향

- 제외 영역은 물리 모델 승인 후 새로운 Flyway 버전 마이그레이션으로 추가한다.
- 이미 공유 환경에 적용된 1차 마이그레이션 파일은 수정하지 않고 후속 버전으로 보정한다.
- 프로젝트 계약ID는 계약 물리 모델이 추가되기 전까지 애플리케이션 참조로 관리한다.
