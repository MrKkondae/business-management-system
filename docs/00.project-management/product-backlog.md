# 제품 백로그 (Product Backlog)

## 1. 문서 개요

본 문서는 프로젝트 관리 시스템의 기능 요구사항 및 개선사항을 관리하기 위한 제품 백로그이다.

백로그는 프로젝트 진행 중 지속적으로 추가, 변경 및 우선순위 조정이 가능하며, 개발 완료 시까지 지속적으로 관리한다.

---

## 2. 개발 로드맵

### v1.0 (MVP)

**목표**

회사에서 실제 사용할 수 있는 최소 기능을 제공한다.

#### 주요 기능

* 로그인, 사용자·역할·메뉴·공통코드·조직 관리
* 고객사·고객담당자·영업활동 관리
* 영업기회 및 수동 사업공고 관리
* 인력투입용 최소 프로젝트 기준정보
* 인력 관리

사용자별 대시보드는 고객·영업·프로젝트·인력 등 원천 업무기능을 구현하고 실제 제공 지표를 확정한 뒤 후속으로 설계·구현한다. v1.0에서는 로그인 후 현재 구현되어 있고 사용자에게 접근권한이 있는 첫 업무 메뉴로 이동한다.

#### v1.0 프로젝트 범위 결정

v1.0에서는 독립적인 프로젝트관리 기능 전체를 구현하지 않고, 인력투입에 필요한 최소 프로젝트 기준정보만 포함한다.

| 구분 | 범위 |
| --- | --- |
| 포함 | 프로젝트 기본정보 수동 등록·조회·수정, 프로젝트 기간·상태 관리, 인력투입에서 프로젝트 조회 및 필수 참조 |
| 제외 | 프로젝트 일정, 이슈, 위험, 산출물, 진척률, 프로젝트 원가·손익 및 독립 프로젝트 운영관리 |
| 후속 연계 | v1.1 계약관리 도입 시 계약에서 프로젝트 기준정보를 생성·연결하고 기존 수동 등록 프로젝트의 계약 연결 방식을 확정 |

최소 프로젝트 기준정보는 계약관리 없이도 v1.0 인력투입 기능을 사용할 수 있도록 수동 등록을 허용한다. 인력투입에서 참조 중인 프로젝트는 물리삭제하지 않으며 상태 변경 또는 논리삭제로 관리한다.

---

### v1.1

**목표**

프로젝트 운영 기능을 강화한다.

#### 주요 기능

* 계약 관리
* 매출 관리
* 청구 및 수금 관리
* 알림 기능
* 권한 관리 고도화
* 사업공고 자동수집·검토·재처리

---

### v1.2

**목표**

사용 편의성을 개선한다.

#### 주요 기능

* 메모 기능
* 첨부파일 관리
* 통합 검색
* 엑셀 다운로드
* 프로젝트 통계

---

### v2.0

**목표**

AI 기반 업무 지원 기능을 제공한다.

#### 주요 기능

* AI 문서 생성
* AI 회의록 요약
* AI 일정 추천
* AI 프로젝트 위험도 분석
* AI 보고서 자동 생성
