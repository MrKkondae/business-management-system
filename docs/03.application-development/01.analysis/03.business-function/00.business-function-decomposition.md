# 업무기능분해도 (Business Function Decomposition)

## 1. 문서 개요

본 문서는 BMS(Business Management System)의 업무기능을 계층적으로 정의한 문서이다.

업무기능분해도(BFD)는 시스템에서 제공하는 업무영역과 업무기능의 계층구조를 정의하며, 요구사항 정의서, 업무 프로세스, 메뉴구조도, 화면목록, API 목록, DB 설계 및 테스트케이스의 기준 문서로 활용한다.

본 문서는 **업무영역(L1)과 업무기능(L2)** 수준까지 정의하며, 세부 기능(L3, L4)은 「상세 업무기능분해도」 문서에서 관리한다.

# 3. 업무기능 간 연계

업무기능은 다음과 같은 흐름으로 연계된다.

```text
고객관리
   ↓
영업관리
   ↓
계약관리
   ↓
프로젝트 기준정보 생성
   ↓
인력관리
   ↓
예산·원가관리
   ↓
매출·수금관리

```

---

# 4. 업무기능 정의 원칙

* 업무기능은 업무영역(L1)과 업무기능(L2) 수준으로 정의한다.
* 업무기능별 세부 기능(L3, L4)은 「상세 업무기능분해도」 문서에서 관리한다.
* 하나의 업무기능은 하나 이상의 메뉴, 화면, API 및 데이터 모델과 연계될 수 있다.
* 업무기능 변경 시 요구사항 정의서, 업무 프로세스, 메뉴구조도 및 화면목록을 함께 검토하여 일관성을 유지한다.
