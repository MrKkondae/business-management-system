# 화면목록

## 1. 문서 개요

### 1.1 목적

본 문서는 BMS(Business Management System)의 화면 구성을 정의한다.

화면목록은 메뉴구조도 및 상세 업무기능분해도를 기반으로 사용자에게 제공되는 화면을 정의하며, 화면설계서, API 목록 및 테스트케이스의 기준으로 활용한다.

---

### 1.2 화면 ID 규칙

화면 ID는 업무영역별 Prefix를 사용한다.

| Prefix | 업무영역 |
|---------|----------|
| COM | 공통 |
| SYS | 시스템관리 |
| CUS | 고객관리 |
| SAL | 영업관리 |
| CON | 계약관리 |
| RES | 인력관리 |
| CST | 예산·원가관리 |
| REV | 매출·수금관리 |

프로젝트관리와 유지관리는 구현범위에서 제외하므로 화면 Prefix와 화면 ID를 부여하지 않는다.
