# 시스템관리 상세 업무기능분해도

## 1. 문서 개요

### 1.1 목적

본 문서는 BMS(Business Management System)의 시스템관리 업무에 대한 상세 업무기능을 정의한다.

시스템관리 기능은 사용자 및 권한 관리, 메뉴 관리, 공통코드 관리, 조직 관리 등 시스템 운영에 필요한 기본 기능을 제공하며, 시스템 운영의 기준정보를 관리한다.

# 4. 업무 연계

시스템관리 기능은 다음 업무와 연계된다.

| 업무      | 연계 내용                             |
| ------- | --------------------------------- |
| 공통      | 사용자 인증, 메뉴 및 권한정보를 제공한다.          |
| 고객관리    | 메뉴 및 기능 접근권한을 제어한다.               |
| 영업관리    | 메뉴 및 기능 접근권한을 제어한다.               |
| 계약관리    | 메뉴 및 기능 접근권한을 제어한다.               |
| 프로젝트 기준정보 참조 업무 | 조직, 직원 및 사용자 정보를 제공하고 접근권한을 제어한다. |
| 인력관리    | 내부 직원정보와 사용자 계정정보를 연계한다. 외주인력은 사용자 계정과 연계하지 않는다. |
| 예산·원가관리 | 역할별 조회 및 관리 권한을 제어한다.             |
| 매출·수금관리 | 역할별 조회 및 관리 권한을 제어한다.             |
