# 상세 업무기능분해도 (Business Function Decomposition Detail)

## 1. 문서 개요

### 1.1 목적

본 문서는 BMS(Business Management System)의 상세 업무기능을 정의하는 문서이다.

업무기능분해도(BFD)에서 정의한 업무영역(L1) 및 업무기능(L2)을 기준으로 업무별 상세기능(L3, L4)을 정의하며, 시스템 구축 시 기능 구현의 기준으로 활용한다.

본 문서는 메뉴구조도, 화면목록, 화면설계서, API 목록, DB 설계서 및 테스트케이스의 기준 문서로 활용한다.

---

### 1.2 적용 범위

본 문서는 BFD(Business Function Diagram)에서 정의한 업무영역별 상세 업무기능을 관리한다.

## 3. 기능 ID 부여 규칙

기능 ID는 업무기능분해도(BFD)의 계층 구조를 기준으로 부여한다.

### 3.1 ID 구성

```text
BFD-L1-L2-L3-L4
```

예시)

| 기능 ID           | 설명       |
| --------------- | -------- |
| BFD-01          | 시스템관리    |
| BFD-01-01       | 사용자관리    |
| BFD-01-01-01    | 사용자 기본관리 |
| BFD-01-01-01-01 | 사용자 조회   |
| BFD-01-01-01-02 | 사용자 등록   |
| BFD-01-01-01-03 | 사용자 수정   |
| BFD-01-01-01-04 | 사용자 삭제   |

---

### 3.2 L1 규칙

| 코드     | 업무영역    |
| ------ | ------- |
| BFD-01 | 시스템관리   |
| BFD-02 | 공통      |
| BFD-03 | 고객관리    |
| BFD-04 | 영업관리    |
| BFD-05 | 계약관리    |
| BFD-06 | 프로젝트관리  |
| BFD-07 | 인력관리    |
| BFD-08 | 예산·원가관리 |
| BFD-09 | 매출·수금관리 |
| BFD-10 | 유지관리    |

---

### 3.3 L2 규칙

L2는 업무영역별 주요 업무기능을 정의한다.

예시)

```text
BFD-03-01 고객사관리

BFD-03-02 고객담당자관리

BFD-03-03 영업활동관리
```

---

### 3.4 L3 규칙

L3는 하나의 업무를 구성하는 기능 그룹을 정의한다.

예시)

```text
사용자관리

사용자권한관리

비밀번호관리
```

---

### 3.5 L4 규칙

L4는 시스템에서 제공하는 실제 기능 단위를 정의한다.

예시)

```text
사용자조회

사용자등록

사용자수정

사용자삭제

비밀번호초기화
```

## 4. 작성 원칙

상세 업무기능은 다음 기준에 따라 작성한다.

### 4.1 기능 계층

* L1 : 업무영역
* L2 : 업무기능
* L3 : 기능그룹
* L4 : 기능

---

### 4.2 기능 정의 기준

L4는 화면 또는 API 단위의 기능을 기준으로 정의한다.

예)

* 조회
* 등록
* 수정
* 삭제
* 승인
* 반려
* 복사
* 엑셀 업로드
* 엑셀 다운로드
* 출력
* 상태 변경

---

### 4.3 작성 기준

* 하나의 기능은 하나의 업무목적을 가진다.
* 기능명은 업무 용어를 사용한다.
* 동일한 기능명은 동일한 의미로 사용한다.
* 기능 ID는 변경하지 않는 것을 원칙으로 한다.
* 기능이 삭제되더라도 기존 ID는 재사용하지 않는다.
