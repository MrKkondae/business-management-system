# 공통 상세 업무기능분해도

## 1. 문서 개요

### 1.1 목적

본 문서는 BMS(Business Management System)의 여러 업무영역에서 공통으로 사용하는 첨부파일, 메모 및 알림 기능을 정의한다.

공통 기능은 인력 경력 증빙, 영업활동 등 개별 업무 화면에서 재사용하며 독립된 업무 데이터를 중복 생성하지 않는다.

### 1.2 적용 범위

- 업무대상별 첨부파일 등록, 조회, 다운로드, 삭제 및 연결
- 업무대상별 메모 조회, 등록, 수정 및 삭제
- 사용자별 알림 조회 및 읽음 처리
- 첨부파일 저장 및 접근에 필요한 기본 검증

# 4. 주요 업무 규칙

| 구분 | 업무 규칙 |
| --- | --- |
| 업무대상 식별 | 첨부파일과 메모는 업무구분과 업무ID의 조합으로 대상 업무를 식별한다. 다형 참조이므로 대상 존재 여부는 애플리케이션에서 검증한다. |
| 파일 크기 | 시스템에서 설정한 최대 파일 크기를 초과하는 파일은 등록할 수 없다. |
| 파일 확장자 | 시스템에서 허용한 확장자만 등록할 수 있으며 파일명 확장자와 실제 파일 형식을 함께 검증한다. 실행 파일 등 금지 형식은 등록할 수 없다. |
| 저장경로 | 저장경로와 저장파일명은 시스템이 생성하며 사용자가 직접 지정할 수 없다. 경로 정규화 후 허용된 저장영역 밖의 경로는 사용할 수 없다. |
| 접근권한 | 첨부파일 조회·다운로드·등록·삭제 권한은 연결된 업무대상의 조회·관리 권한을 따른다. |
| 파일 삭제 | 계약문서 등 다른 업무정보가 파일을 참조하고 있으면 삭제를 차단하고 참조 해제 후 삭제하도록 안내한다. |
| 메모 권한 | 메모 조회·등록 권한은 대상 업무 권한을 따르며 수정·삭제는 작성자 또는 관리권한이 있는 사용자만 가능하다. |
| 알림 권한 | 알림은 수신자 본인만 조회하고 읽음 처리할 수 있다. |

# 5. 예외 및 오류처리 기준

| 상황 | 처리 기준 |
| --- | --- |
| 최대 파일 크기 초과 | 등록을 차단하고 허용 가능한 최대 크기를 안내한다. |
| 허용되지 않은 확장자 또는 파일 형식 | 등록을 차단하고 허용 가능한 파일 형식을 안내한다. |
| 잘못된 저장경로 | 파일 저장을 중단하고 보안 오류로 기록한다. |
| 업무대상 없음 | 첨부파일 연결 또는 메모 등록을 차단한다. |
| 업무대상 접근권한 없음 | 파일 조회·다운로드·등록·삭제 및 메모 처리를 차단한다. |
| 참조 중인 파일 삭제 | 파일을 참조하는 업무정보가 있음을 안내하고 삭제를 차단한다. |
| 다른 사용자의 알림 처리 | 알림 조회 및 읽음 처리를 차단한다. |
