# 시스템관리 상세 업무기능분해도

## 1. 문서 개요

### 1.1 목적

본 문서는 BMS(Business Management System)의 시스템관리 업무에 대한 상세 업무기능을 정의한다.

시스템관리 기능은 사용자 및 권한 관리, 메뉴 관리, 공통코드 관리, 조직 관리 등 시스템 운영에 필요한 기본 기능을 제공하며, 시스템 운영의 기준 정보를 관리한다.

# 4. 업무 연계

시스템관리 기능은 다음 업무와 연계된다.

| 업무     | 연계 내용          |
| ------ | -------------- |
| 고객관리   | 사용자 권한 검증      |
| 영업관리   | 메뉴 및 권한 제어     |
| 계약관리   | 사용자 권한 제어      |
| 프로젝트관리 | 조직 및 사용자 정보 활용 |
| 인력관리   | 직원 기본정보 연계     |
| 공통     | 로그인 및 권한 관리    |
