# 고객관리 상세 업무기능분해도

## 1. 문서 개요

### 1.1 목적

본 문서는 BMS(Business Management System)의 고객관리 업무에 대한 상세 업무기능을 정의한다.

고객관리 기능은 고객사 및 고객 담당자의 기본정보를 관리하고, 영업활동 및 거래이력을 통합 관리하여 영업관리, 계약관리 및 프로젝트관리의 기준 정보를 제공한다.

## 3.3 영업활동관리

영업활동은 고객과의 모든 활동을 통합 관리한다.

활동 유형은 **공통코드(ACTIVITY_TYPE)** 로 관리하며 방문, 전화, 메일, 미팅, 제안, 기타 등을 선택할 수 있다.

### 3.3.1 영업활동관리

| 기능ID            | 기능명     | 설명             |
| --------------- | ------- | -------------- |
| BFD-03-03-01-01 | 영업활동 조회 | 고객 영업활동을 조회한다. |
| BFD-03-03-01-02 | 영업활동 등록 | 영업활동을 등록한다.    |
| BFD-03-03-01-03 | 영업활동 수정 | 영업활동을 수정한다.    |
| BFD-03-03-01-04 | 영업활동 삭제 | 영업활동을 삭제한다.    |

#### 활동유형(공통코드)

| 코드       | 설명 |
| -------- | -- |
| VISIT    | 방문 |
| CALL     | 전화 |
| MEETING  | 미팅 |
| EMAIL    | 메일 |
| PROPOSAL | 제안 |
| ETC      | 기타 |

## 3.4 거래이력관리

거래이력은 고객사와 관련된 계약, 프로젝트, 유지보수 등의 업무이력을 통합 조회한다.

### 3.4.1 거래이력관리

| 기능ID            | 기능명       | 설명                  |
| --------------- | --------- | ------------------- |
| BFD-03-04-01-01 | 거래이력 조회   | 고객사의 전체 거래이력을 조회한다. |
| BFD-03-04-01-02 | 거래이력 상세조회 | 거래이력의 상세정보를 조회한다.   |

#### 거래이력 조회 대상

* 영업기회
* 제안
* 계약
* 프로젝트
* 유지보수

# 4. 업무 연계

고객관리 기능은 다음 업무와 연계된다.

| 업무     | 연계 내용                 |
| ------ | --------------------- |
| 영업관리   | 영업기회 및 제안 대상 고객 정보 제공 |
| 계약관리   | 계약 대상 고객 정보 제공        |
| 프로젝트관리 | 프로젝트 발주기관 정보 제공       |
| 유지관리   | 유지보수 대상 고객 정보 제공      |
| 공통     | 고객 검색 및 고객 정보 조회      |
